Design Philosophy

  • Obsidian foundation: Dark base makes the 3D viewport and colorful assets pop
  • Geometric language: Chamfered corners, diamond shapes, faceted surfaces throughout
  • Gem palette: Each category has a gem color (citrine, ruby, emerald, sapphire, amethyst)
  • The Forge: Asset creation feels like crafting, not just "generating"
  • Professional structure: Clean hierarchy, readable text, functional layout
  • Bold but not loud: Color is intentional, not overwhelming
